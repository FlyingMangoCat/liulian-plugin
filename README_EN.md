### Description
`liulian-plugin` is a Yunzai-Bot plugin that provides extended features including Genshin Impact underground maps, Bilibili push notifications, plugin management, and group chat functions such as Genshin/HSR character guessing, marry group members, Liulian AI, random expressions, horoscope, chat statistics, weather, multi-group broadcasts, and group muting. ### Installation

#### Git Installation (Recommended)
Place liulian-plugin in the plugins directory of Yunzai-Bot, restart Yunzai-Bot, and it will be ready to use.

Please use git for installation to facilitate future upgrades. Open a terminal in the Yunzai-Bot root directory and run one of the following commands:

#### Install via Gitee
```
git clone https://gitee.com/huifeidemangguomao/liulian-plugin.git ./plugins/liulian-plugin/
pnpm install -P
```

#### Install via GitHub
```
git clone https://github.com/flyingmangocat/liulian-plugin.git ./plugins/liulian-plugin/
pnpm install -P
```
